# AI Corpus Lab

Experimental work on historical corpora, text processing, semantic search and AI-assisted analysis.

## Current focus

- Building cleaned text corpora from OCR and TXT sources
- Splitting long texts into searchable chunks
- Creating metadata for documents
- Testing semantic search with embeddings
- Exploring retrieval-augmented generation for historical and conceptual analysis

## Projects in progress

### 1. Historical corpus pipeline

A pipeline for turning raw historical texts into structured datasets.

Steps:
- text cleaning
- metadata attribution
- sentence extraction
- chunking
- embedding
- semantic retrieval

### 2. Strategic data analysis

Experiments with public datasets, especially event data, for geopolitical and strategic reading.

## Status

Work in progress.
